# efsm-copy

Skill de copywriting para a equipe EFSM. Cria textos persuasivos para Landing Pages, anúncios no Meta Ads, artigos de blog e postagens em redes sociais.

## Como usar

```
/efsm-copy
```

Após invocar, o Claude vai coletar as informações necessárias antes de produzir qualquer copy.

## O que essa skill cobre

| Formato | Método aplicado |
|--------|----------------|
| Landing Page | AIDA, PAS ou 4U's (Claude escolhe o melhor) |
| Anúncio Meta Ads | PAS com bullets e CTA com gatilho mental |
| Artigo de Blog | SEO-friendly, >30% palavras de transição, <10% voz passiva |
| Post Redes Sociais | Adaptado por plataforma (Instagram, LinkedIn, Facebook) |

## Instalação

```bash
cp skill.md ~/.claude/skills/efsm-copy.md
```
