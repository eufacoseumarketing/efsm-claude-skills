# efsm-social

Skill de social media para a equipe EFSM. Cobre estratégia, calendário editorial, análise de planejamento, auditoria de perfil e interpretação de métricas — com boas práticas específicas para os nichos atendidos pela agência.

## Como usar

```
/efsm-social
```

Após invocar, descreva o que precisa. A skill identifica o modo automaticamente.

## O que essa skill cobre

| Modo | Quando usar |
|------|------------|
| **Calendário Editorial** | Criar ou completar um plano de conteúdo para o mês |
| **Análise de Planejamento** | Revisar um planejamento antes da aprovação interna |
| **Auditoria de Perfil** | Diagnosticar o perfil de um cliente novo ou em renovação |
| **Análise de Métricas** | Interpretar dados e montar relatório de desempenho |
| **Estratégia** | Definir posicionamento, pilares e frequência para um cliente |
| **Produção de Conteúdo** | Criar roteiro de Reel ou estrutura de carrossel |

## Nichos cobertos

- Jurídico (advocacia) — com restrições OAB
- Saúde / Medicina — com restrições CFM
- B2B Tech / SaaS
- Varejo / Decoração
- Turismo Premium

## Instalação

**macOS / Linux**
```bash
bash scripts/install.sh efsm-social
```

**Windows**
```powershell
.\scripts\install.ps1 efsm-social
```
